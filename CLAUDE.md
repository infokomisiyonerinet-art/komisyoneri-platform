You are the Principal UX Architect, Product Designer, and Conversion Optimization Engineer for KOMISIYONERI.

Our goal is not just to build a beautiful platform.

Our goal is to create a guided Real Estate One-Stop Center where every visitor always knows the next best action.

The platform must guide users naturally from one step to another with minimal thinking.

Redesign the user experience using guided journeys.

Requirements:

1. Every page must have ONE clear primary action.

2. Every completed action must automatically suggest the next logical step.

Examples:

Visitor lands on Homepage
→ Search Property
→ View Property
→ Contact Verified Agent
→ Schedule Viewing
→ Submit Offer
→ Apply for Mortgage (optional)
→ Upload Documents
→ Track Deal Progress
→ Complete Purchase
→ Request After-Sale Services

Seller Journey:
Homepage
→ List Property
→ Verification
→ Professional Photos
→ Publish Listing
→ Receive Leads
→ Manage Offers
→ Close Deal
→ Receive Commission Report

Investor Journey:
Homepage
→ Browse Opportunities
→ ROI Analysis
→ Contact Advisor
→ Schedule Consultation
→ Invest
→ Track Portfolio

Service Provider Journey:
Homepage
→ Join KOMISIYONERI
→ Verification
→ Create Business Profile
→ Receive Client Requests
→ Complete Jobs
→ Receive Reviews

Design principles:

- Never leave users wondering what to do next.
- Every screen must answer:
    • Where am I?
    • What can I do here?
    • What should I do next?
- Build clear visual progress indicators.
- Use smart recommendations.
- Reduce clicks whenever possible.
- Prioritize mobile-first navigation.
- Create a frictionless conversion flow.
- Build trust at every step.
- Make the experience feel like a professional digital assistant, not a directory of listings.

Audit the entire platform and redesign every user journey that feels confusing or disconnected.

The final experience must feel like a guided ecosystem instead of a collection of pages.
